# hoichoi-downloader
hoichoi-downloader using NodeJs

## Hoichoi Downloader
"hoichoi-downloader" is a simple downloader to download hoichoi series.

## Usages

```
node app [series_browser_url]
```

## Examples
```
node app https://www.hoichoi.tv/show/title/eken-babu-web-series
```
